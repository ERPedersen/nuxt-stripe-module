## Setup
- Add `nuxt-stripe-module` dependency using yarn or npm to your project
- Add `nuxt-stripe-module` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-stripe-module',

    // With options
    ['nuxt-stripe-module', { /* module options */ }],
 ]
}
```
